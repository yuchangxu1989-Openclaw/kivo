## 一、核心详写测试用例

### TC-A01-AC1
- **ID**：TC-A01-AC1
- **关联 FR-AC**：FR-A01 / AC1
- **前置条件**：系统已接入一段包含事实、方法论、决策、经验、意图、元认知表达的对话样本。
- **操作步骤**：
  1. 导入对话样本。
  2. 触发对话知识提取流程。
  3. 查看提取结果列表。
- **预期结果**：系统识别并输出六类知识，且每条知识被正确标注为 fact、methodology、decision、experience、intent、meta 中的一类。
